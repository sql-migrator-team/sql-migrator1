About SQL Migrator

SQL Migrator is a backend-only ETL platform designed for SQL-to-SQL database migration, schema generation, file import/export operations, validation, reporting, and migration history management. The system exposes REST APIs that allow administrators and authorized users to execute migration workflows between supported database systems while maintaining secure internal records and reports.

Built using Flask and SQLAlchemy, the application follows a clean architecture approach that separates authentication, migration processing, validation, reporting, and database connectivity into dedicated modules. Internal application data is stored in SQLite, while external databases are connected dynamically through API request payloads.

The platform provides user authentication with JWT and role-based access control, SQL migration with datatype mapping, CSV and Excel import/export functionality, schema generation, migration validation, report generation in JSON, Text, and PDF formats, and persistent migration history tracking.

The backend architecture consists of Processors for CSV and Excel handling, Converters for SQL transformation and export operations, Validators for compatibility verification, Database modules for external connections, Reports modules for document generation, Models for internal storage, and REST API Routes for system access.

Supported external database engines include MySQL, PostgreSQL, SQLite, and Oracle. Internal SQLite storage maintains users, migrations, migration history, reports, uploaded files, exported files, logs, and system configuration records.

The technology stack includes Python 3.12+, Flask, Flask-RESTful, Flask-JWT-Extended, SQLAlchemy, Bcrypt, Pandas, OpenPyXL, PyMySQL, psycopg2-binary, cx_Oracle, FPDF, and Pytest. This modular design provides a scalable, maintainable, and extensible foundation for enterprise-grade database migration workflows.
